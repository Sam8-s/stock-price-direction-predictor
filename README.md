## Stock Price Direction Predictor

Predicts whether a stock price will go UP or DOWN the next day using Machine Learning and technical indicators.

## Setup

## 1 Clone the repository

git clone https://github.com/Sam8-s/stock-price-direction-predictor.git
cd stock-price-direction-predictor

## 2 Install dependencies

pip install -r requirements.txt

## 3 Download dataset

python download_data.py

## 4 Run the dashboard

python -m streamlit run app.py
## Dataset
https://www.kaggle.com/datasets/jacksoncrow/stock-market-dataset
